# Yet-Another-yt-dlp-Frontend

Simple yt-dlp frontend made with Pyside6.

Tested on Debian Trixie and CachyOS. I may do a build for Windows.

If you double click the item on the table it will make a small windows with the stdout of that process, everything else should be 'intuitive', graphic design is my passsion. Check the settings for the download location.
